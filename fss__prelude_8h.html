#define kLambda   16

For security param lambda in papers, it is lambda / 8.

◆ kParallelDepth

#define kParallelDepth   1

Parallel degree of full domain eval.

<= 0 disables parallelization.

‍0 gives 2 ** kParallelDepth concurrent tasks running on a fixed num of threads set by OpenMP.

Higher is not better, particularly when > physical core num with hyper-threading, and users need trials. For kLambda = 16, 1 already results in top performance.
